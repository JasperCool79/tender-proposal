## How to Run the Project

## Git Clone
```
git clone https://github.com/JasperCool79/tender-proposal.git
```
## Compoer Instal
```
composer install
```

## .env setup
```
change .env.example to .env
```
## Key Generate
```
php artisan key:generate
```
## db setup
```
create new db with the name "laravel" in phpMyadmin MySQL and change the db name at .env
```
## npm 
```
npm install
```
## npm 
```
npm run dev
```
### Migration
```
php artisan migrate --seed
```
### Start Serve
```
php artisan serve
```
